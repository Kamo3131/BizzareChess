# BizzareChess
Game of chess with personal twist

## How to run?
In build directory\
`cmake .. -G "Ninja"`\
`cmake --build .`\
`.\tests\chess_tests` (Optional)\
`.\chess_logic`\
In main project directory you can just use my initializer.ps1:\
`.\initializer.ps1`\
It will run all the needed commands. You just have to run `.\chess_logic` after it finishes.
